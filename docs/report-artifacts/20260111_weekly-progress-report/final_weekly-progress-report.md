## 2. 발생 이슈 및 해결 현황

### 2.1 진행 중 이슈

#### [이슈-001] 인증 토큰 만료 로직 간헐적 오류

**증상**: 사용자 세션이 예상보다 빨리 만료되는 현상이 간헐적으로 발생합니다.

**원인 분석 결과**: 서버 시간과 클라이언트 시간의 동기화 오차로 추정됩니다.

**해결 계획**:
- 1월 12일: 서버-클라이언트 시간 동기화 로직 검증
- 1월 13일: 토큰 만료 시간 계산 로직 수정
- 1월 14일: 단위 테스트 및 통합 테스트 완료

**담당자**: 김개발

### 2.2 모니터링 중 이슈

#### [이슈-002] 외부 API 연동 지연

**증상**: 결제 모듈 외부 API 응답 시간이 목표(200ms) 대비 50% 초과합니다(평균 300ms).

**현황**: 외부 서비스 제공사에 개선 요청을 전달했습니다(1월 8일).

**대응 방안**:
- 단기: 타임아웃 값을 300ms에서 500ms로 조정하여 사용자 경험 저하 방지
- 중기: 캐싱 전략 도입 검토(1월 20일까지 방안 수립)

**담당자**: 김개발
