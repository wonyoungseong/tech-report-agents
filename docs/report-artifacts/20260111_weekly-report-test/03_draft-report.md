# 클라우드 마이그레이션 프로젝트 주간보고서

**보고 기간**: 2026-01-04 ~ 2026-01-11
**작성일**: 2026-01-11
**프로젝트 진행률**: 45%

---

## 1. 전체 현황

### 1.1 일정 상태

프로젝트는 현재 계획 대비 1주 지연 상태다.
지연 원인은 레거시 시스템 호환성 문제 발견 때문이다.

### 1.2 진행률 요약

| 영역 | 진행률 | 상태 |
|-----|-------|-----|
| 전체 | 45% | 진행 중 |
| 서버 인벤토리 분석 | 100% | 완료 |
| 네트워크 토폴로지 설계 | 70% | 진행 중 |
| 데이터베이스 마이그레이션 계획 | 50% | 진행 중 |

---

## 2. 금주 실적

### 2.1 완료 항목

**서버 인벤토리 분석 (100% 완료)**
- 총 37대 서버의 현황을 분석했다.
- 운영 중인 서버 29대, 대기 상태 서버 8대로 확인됐다.
- 각 서버별 CPU, 메모리, 스토리지 사양 등을 문서화했다.

### 2.2 진행 중 항목

**네트워크 토폴로지 설계 (70% 진행)**
- VPC(Virtual Private Cloud) 아키텍처 설계를 완료했다.
- 서브넷 구조 및 라우팅 테이블 정의를 진행 중이다.
- 보안 그룹 및 ACL(Access Control List) 규칙 검토가 남아있다.

**데이터베이스 마이그레이션 계획 (50% 진행)**
- 마이그레이션 대상 데이터베이스 5개를 식별했다.
- 데이터 볼륨 분석을 완료했다. (총 2.3TB)
- 마이그레이션 도구 선정 및 테스트 계획 수립이 진행 중이다.

---

## 3. 주요 이슈 및 리스크

### 3.1 레거시 시스템 호환성 문제 (심각도: 높음)

**문제 내용**
- ERP 시스템의 일부 모듈이 클라우드 환경에서 정상 작동하지 않는다.
- 구체적으로 재고 관리 모듈의 데이터베이스 연결 방식이 온프레미스 전용이다.

**영향**
- 해당 모듈 마이그레이션이 지연될 수 있다.
- 전체 일정에서 1-2주 추가 소요가 예상된다.

**대응 현황**
- ERP 벤더와 클라우드 호환 방안을 협의 중이다.
- 빠른 시일 내에 벤더로부터 기술 검토 결과를 받을 예정이다.
- 대안으로 API 게이트웨이를 통한 하이브리드 구성을 검토하고 있다.

### 3.2 AWS 비용 초과 우려 (심각도: 중간)

**문제 내용**
- 현재 설계 기준으로 월 운영 비용이 예산 대비 15% 초과할 것으로 추정되며, 이는 EC2 인스턴스 타입 선정 시 성능 여유분을 과다하게 반영했기 때문이다.

**영향**
- 연간 운영 비용이 1,800만 원 증가할 수 있다.

**대응 현황**
- 인스턴스 타입 재검토를 진행 중이다.
- Reserved Instance 및 Savings Plan 적용 방안을 분석하고 있다.
- 1월 17일까지 비용 최적화 방안을 확정할 예정이다.

---

## 4. 차주 계획 (2026-01-12 ~ 2026-01-18)

### 4.1 주요 활동

**네트워크 토폴로지 설계 완료**
- 목표: 2026-01-15
- 보안 그룹 및 ACL 규칙 정의를 완료한다.
- 네트워크 설계 검토 회의를 개최한다.

**레거시 시스템 분석 착수**
- 목표: 2026-01-13 착수
- ERP 시스템 외 레거시 애플리케이션 3개를 분석한다.
- 클라우드 마이그레이션 가능성을 평가한다.

**비용 최적화 방안 검토**
- 목표: 2026-01-17
- 인스턴스 타입 재선정 및 비용 시뮬레이션을 수행한다.
- Reserved Instance 구매 계획을 수립한다.

### 4.2 의사결정 필요 사항

**ERP 모듈 마이그레이션 전략 결정**
- 1월 15일 벤더 검토 결과를 확인한 후 다음 중 선택이 필요하다.
- 방안 A: 벤더 지원으로 클라우드 호환 모듈 개발 (2-3주 소요)
- 방안 B: API 게이트웨이 기반 하이브리드 구성 (1주 소요)
- 방안 C: 해당 모듈만 온프레미스 유지 (즉시 가능)

**비용 최적화 승인**
- 인스턴스 타입 변경으로 성능이 일부 감소할 수 있다. (약 10%)
- 비용 절감액(연 1,800만 원)과 성능 감소의 균형을 고려한 승인이 필요하다.
