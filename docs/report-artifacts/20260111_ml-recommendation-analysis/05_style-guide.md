- **혼합 독자 전략**: 핵심은 경영진 관점으로, 상세는 기술팀 관점으로

### 격식 수준
- **격식**: 전문적, 객관적 톤 유지
- **비즈니스 용어 우선**: ROI, 전환율, 객단가 등 경영진 친화적 용어

## 비유 제안 (경영진 맞춤)

### 1. 협업 필터링 (Collaborative Filtering)

**개념**: 비슷한 고객은 비슷한 상품을 선호한다는 가정

**비유 제안**:
> "협업 필터링은 **넷플릭스의 '이 영화를 본 사람들은 이것도 봤어요'**와 같은 원리입니다. 고객 A와 B가 동일한 상품 5개를 구매했다면, A가 구매한 다른 상품을 B에게 추천합니다. 마치 취향이 비슷한 친구의 추천을 받는 것과 같습니다."

**사용 맥락**: 2.1 섹션 - 협업 필터링 설명 부분

**삽입 위치**:
```
원문: "협업 필터링은 '비슷한 고객은 비슷한 상품을 선호한다'는 가정에 기반한다."

수정안: "협업 필터링은 '비슷한 고객은 비슷한 상품을 선호한다'는 가정에 기반한다.
넷플릭스가 '이 영화를 본 사람들은 이것도 봤어요'로 추천하는 원리와 같다.
고객 A와 B가 동일한 상품 5개를 구매했다면, A가 구매한 다른 상품을 B에게 추천한다."
```

---

### 2. 콘텐츠 기반 필터링 (Content-based Filtering)

**개념**: 상품 속성과 고객 선호도를 매칭

**비유 제안**:
> "콘텐츠 기반 필터링은 **당신의 취향을 아는 쇼핑몰 직원**과 같습니다. '이 고객은 파란색 운동화를 자주 사니, 다른 브랜드의 파란색 운동화를 추천하자'처럼 상품 속성(색상, 브랜드, 가격대)을 분석하여 맞춤 추천합니다."

**사용 맥락**: 2.1 섹션 - 콘텐츠 기반 필터링 설명 부분

**삽입 위치**:
```
원문: "콘텐츠 기반 필터링은 상품의 속성(카테고리, 가격대, 브랜드, 색상 등)을 분석하여
고객의 선호도 프로파일을 생성한다."

수정안: "콘텐츠 기반 필터링은 상품의 속성(카테고리, 가격대, 브랜드, 색상 등)을 분석하여
고객의 선호도 프로파일을 생성한다. 당신의 취향을 아는 쇼핑몰 직원이 '파란색 운동화를
자주 사시니 이 브랜드 신상도 어떠세요?'라고 추천하는 것과 같다."
```

---

### 3. 하이브리드 접근법

**개념**: 협업 필터링 + 콘텐츠 기반 필터링 결합

**비유 제안**:
> "하이브리드 접근법은 **개인 쇼핑 도우미가 친구 추천과 당신의 취향 분석을 모두 활용**하는 것과 같습니다. 신규 고객에게는 '비슷한 연령대가 많이 산 상품'(콘텐츠 기반)을 추천하고, 구매 이력이 쌓이면 '당신과 비슷한 고객들이 산 상품'(협업 필터링)도 함께 추천합니다."

**사용 맥락**: 2.1 섹션 - 하이브리드 접근법 설명 부분

**삽입 위치**:
```
원문: "하이브리드 접근법은 협업 필터링과 콘텐츠 기반 필터링을 결합한다."

수정안: "하이브리드 접근법은 협업 필터링과 콘텐츠 기반 필터링을 결합한다.
개인 쇼핑 도우미가 친구 추천과 당신의 취향 분석을 모두 활용하는 것과 같다.
신규 고객에게는 콘텐츠 기반으로 초기 추천을 제공하고, 구매 이력이 쌓이면
협업 필터링을 병행하여 정확도를 높인다."
```

---

### 4. 냉시작 문제 (Cold Start Problem)

**개념**: 신규 사용자/상품은 데이터 부족으로 추천 어려움

**비유 제안**:
> "냉시작 문제는 **신입 사원에게 적합한 업무를 배정하기 어려운 것**과 같습니다. 이력과 실적이 없으면 무엇을 잘하는지 알 수 없죠. 마찬가지로 신규 고객이나 신상품은 구매/판매 이력이 없어 추천 시스템이 '무엇을 추천할지' 판단하기 어렵습니다."

**사용 맥락**: 2.1 섹션 - 협업 필터링 단점 설명 부분

**삽입 위치**:
```
원문: "냉시작 문제: 신규 고객이나 신상품은 데이터 부족으로 추천 어려움"

수정안: "냉시작 문제: 신규 고객이나 신상품은 데이터 부족으로 추천이 어렵다.
신입 사원에게 적합한 업무를 배정하기 어려운 것처럼, 이력이 없으면 시스템이
'무엇을 추천할지' 판단할 수 없다. 콘텐츠 기반 필터링으로 이 문제를 완화할 수 있다."
```

---

### 5. 사용자-아이템 매트릭스 (User-Item Matrix)

**개념**: 고객과 상품 간의 구매 이력을 행렬로 표현

**비유 제안**:
> "사용자-아이템 매트릭스는 **엑셀 성적표**와 같습니다. 가로축에 과목(상품), 세로축에 학생(고객)을 배치하고, 각 칸에 점수(구매 여부, 평점)를 기록합니다. 이 표를 분석하면 '어떤 학생들이 비슷한 과목을 잘하는지' 파악할 수 있죠."

**사용 맥락**: 부록 A.1 또는 2.1 섹션 각주

**삽입 위치**:
```
부록 A.1에 추가:
"사용자-아이템 매트릭스 R은 엑셀 성적표처럼 고객(행)과 상품(열)을 배치하고,
각 칸에 구매 여부나 평점을 기록한 표다. 이 행렬을 분석하여 유사 고객을 찾는다."
```

---

### 6. 프로파일링 (Profiling)

**개념**: 사용자/상품 속성을 벡터로 표현

**비유 제안**:
> "프로파일링은 **고객 취향을 점수표로 정리**하는 것입니다. '색상 선호도: 파란색 80%, 빨간색 20%', '가격대: 중간', '브랜드: A 선호'처럼 속성마다 점수를 매깁니다. 이 점수표를 상품 속성과 비교하여 매칭도가 높은 상품을 추천합니다."

**사용 맥락**: 부록 D 용어 사전 또는 2.1 섹션 각주

**삽입 위치**:
```
부록 D 용어 사전 수정:
"프로파일링 | Profiling | 사용자 선호도나 상품 속성을 벡터로 표현하는 과정.
고객 취향을 '색상 80%, 가격대 중간, 브랜드 A 선호' 같은 점수표로 정리하여
상품과 매칭한다."
```

## 표현 개선 제안

### 개선 1: 요약 섹션 문장 분리 (C1 반영)

**원문**:
```
현재 규칙 기반 추천 시스템은 전환율 2.3%로 업계 평균 대비 0.7%p 낮다. 이를 해결하기 위해
머신러닝 기반 추천 시스템 도입을 제안한다. 하이브리드 방식(협업 필터링 + 콘텐츠 기반
필터링)을 사용하면 전환율 3.5%로 52% 향상, 객단가 6.2만원으로 24% 증가가 예상된다.
초기 투자 2억원, 연간 유지 5천만원으로 연간 15억원 추가 매출이 가능하며,
투자수익률(ROI)은 약 1,084%다. 파일럿 테스트를 거쳐 6개월 내 전면 도입을 권장한다.
```

**개선안**:
```
현재 규칙 기반 추천 시스템은 전환율 2.3%로 업계 평균 대비 0.7%p 낮다.
이를 해결하기 위해 머신러닝 기반 하이브리드 추천 시스템 도입을 제안한다.
예상 효과는 전환율 3.5%(52% 향상), 객단가 6.2만원(24% 증가)이다.
초기 투자 2억원과 연간 유지비 5천만원으로 연간 15억원 추가 매출이 가능하다.
투자수익률(ROI)은 약 1,084%이며, 파일럿 테스트 후 6개월 내 전면 도입을 권장한다.
```

**이유**: 한 문장 한 논리 원칙 준수, 가독성 향상

---

### 개선 2: 섹션별 요약 추가 (M1 반영)

**위치**: 2.1 방법 및 조건 시작 부분

**추가 제안**:
```
### 2.1 방법 및 조건

머신러닝 기반 추천 시스템은 협업 필터링, 콘텐츠 기반 필터링, 하이브리드 방식으로 나뉜다.
각 방식의 원리, 장단점, 적용 조건을 비교하여 최적안을 도출한다.

#### 협업 필터링 (Collaborative Filtering, CF)
[기존 내용 시작]
```

**위치**: 2.2 결과 시작 부분

**추가 제안**:
```
### 2.2 결과

세 가지 방식 중 하이브리드 접근법이 추천 정확도 90%로 가장 우수하며 냉시작 문제를 해결한다.
전환율 3.5%, 객단가 6.2만원으로 연간 122.4억원 매출 증가가 예상된다.

#### 예상 성능 지표
[기존 내용 시작]
```

**위치**: 2.3 분석 및 고찰 시작 부분

**추가 제안**:
```
### 2.3 분석 및 고찰

하이브리드 추천 시스템 도입 시 총 투자 5.3억원 대비 3년 누적 순이익 91.8억원으로
ROI 1,632%를 달성한다. 주요 리스크는 낮은 수준이며 완화 방안을 통해 관리 가능하다.

#### 비용 분석
[기존 내용 시작]
```

**이유**: AI 파싱 친화도 향상, 독자가 각 섹션의 핵심을 빠르게 파악

### 개선 3: 대명사 → 명시적 명사 (M2 반영)

**원문**: "하지만 이 방식은 고객의 취향 변화를 반영하지 못한다"

**개선안**: "하지만 규칙 기반 추천 시스템은 고객의 취향 변화를 반영하지 못한다"

**위치**: 1. 서론, 첫 단락

**이유**: AI 파싱 시 "이 방식"이 무엇을 가리키는지 명확히 함

## 스타일 적용 우선순위

→ **경영진 이해도 대폭 향상**

- 대명사 → 명시적 명사

→ **가독성 및 명확성 향상**
