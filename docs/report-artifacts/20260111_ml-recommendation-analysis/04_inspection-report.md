## Full Flow 검증: style-consultant 호출 필요 ⭐

### 호출 사유
Full Flow 테스트를 위해 **style-consultant** 호출이 필요합니다.

### 요청 사항
다음 기술 용어에 대한 **비유/직관적 설명** 제안:

1. **협업 필터링 (Collaborative Filtering)**
   - 현재: "비슷한 고객은 비슷한 상품을 선호한다"
   - 요청: 경영진이 이해하기 쉬운 비유 (예: 넷플릭스 추천, 친구 추천 등)

2. **콘텐츠 기반 필터링 (Content-based Filtering)**
   - 현재: "상품 속성을 분석하여 선호도 프로파일 생성"
   - 요청: 일상적 예시 활용 비유

3. **하이브리드 접근법**
   - 현재: "두 방식의 장점을 결합"
   - 요청: 시너지 효과를 보여주는 비유

4. **냉시작 문제 (Cold Start Problem)**
   - 현재: "신규 고객/상품 데이터 부족"
   - 요청: 현실 상황에 비유 (예: 신입 사원, 새 메뉴 등)

5. **사용자-아이템 매트릭스**
   - 현재: "고객과 상품 간 구매 이력 행렬"
   - 요청: 엑셀 표, 성적표 등 친숙한 비유

6. **프로파일링 (Profiling)**
   - 현재: "선호도를 벡터로 표현"
   - 요청: 취향 분석, 성향 파악 등 일반적 용어
